# 🧬 C++ Inheritance Examples

This folder contains well-structured and easy-to-understand examples of
different types of inheritance in C++.

These programs are designed for 2nd Semester CSE students to clearly
understand how inheritance works in Object-Oriented Programming.

------------------------------------------------------------------------

## 📂 Files Included

### 1️⃣ single.cpp

Demonstrates: - Single Inheritance - One base class and one derived
class - Accessing base class members from derived class

------------------------------------------------------------------------

### 2️⃣ multilevel.cpp

Demonstrates: - Multilevel Inheritance - Grandparent → Parent → Child
structure - Inheriting properties across multiple levels

------------------------------------------------------------------------

### 3️⃣ hierarchical.cpp

Demonstrates: - Hierarchical Inheritance - One base class inherited by
multiple derived classes - Code reusability concept

------------------------------------------------------------------------

### 4️⃣ multiple.cpp

Demonstrates: - Multiple Inheritance - One derived class inheriting from
multiple base classes - Accessing members of different base classes

------------------------------------------------------------------------

### 5️⃣ diamond_problem.cpp

Demonstrates: - Diamond Problem - Virtual Inheritance - Avoiding
duplicate base class instances - Proper use of virtual keyword

------------------------------------------------------------------------

## 🧠 Concepts Covered

-   Base class and Derived class
-   Access specifiers (public, protected, private)
-   Code reusability
-   Multiple inheritance structure
-   Virtual inheritance
-   Diamond problem solution

------------------------------------------------------------------------

## 🛠 Compile & Run

Use g++ to compile:

g++ filename.cpp -o output ./output

Example:

g++ single.cpp -o single ./single

------------------------------------------------------------------------

## 🎯 Learning Outcome

After completing these examples, you will understand:

✔ Different types of inheritance in C++\
✔ How derived classes access base class members\
✔ When to use multiple inheritance\
✔ What the diamond problem is\
✔ How virtual inheritance solves duplication issues
