# ⚙️ Advanced Operator Overloading in C++

This folder contains easy-to-understand and well-structured examples of
advanced operator overloading concepts in C++.

These examples are designed for 2nd Semester CSE students to clearly
understand how special operators can be overloaded in classes.

------------------------------------------------------------------------

## 📂 Files Included

### 1️⃣ subscript_operator.cpp

Demonstrates: - Overloading \[\] operator - Returning reference - Array
bounds checking - Modifying object using subscript

------------------------------------------------------------------------

### 2️⃣ function_call_operator.cpp

Demonstrates: - Overloading () operator - Making objects behave like
functions - Functor concept (Function Object)

------------------------------------------------------------------------

### 3️⃣ logical_operator.cpp

Demonstrates: - Overloading logical && operator - Object comparison
logic - Boolean return type

------------------------------------------------------------------------

### 4️⃣ new_delete_operator.cpp

Demonstrates: - Overloading new operator - Overloading delete operator -
Custom memory allocation - Constructor and Destructor behavior

------------------------------------------------------------------------

### 5️⃣ arrow_operator.cpp

Demonstrates: - Overloading -\> operator - Smart pointer concept
(basic) - Accessing member functions using overloaded arrow

------------------------------------------------------------------------

## 🧠 Concepts Covered

-   Unary operator overloading
-   Binary operator overloading
-   Returning references
-   Operator overloading syntax
-   Memory management customization
-   Object behaving like pointer
-   Object behaving like function

------------------------------------------------------------------------

## 🛠 Compile & Run

Use g++ to compile:

g++ filename.cpp -o output ./output

Example:

g++ subscript_operator.cpp -o subscript ./subscript

------------------------------------------------------------------------

## 🎯 Learning Outcome

After completing these examples, you will understand:

✔ How special operators are overloaded\
✔ Difference between prefix and postfix operators\
✔ How objects can mimic arrays, functions, and pointers\
✔ How memory allocation can be customized\
✔ How operator overloading improves code readability
