# 📘 C++ Basics Programs

This folder contains fundamental C++ programs designed for 1st and 2nd
Semester CSE students.

These programs strengthen core programming concepts such as loops,
conditionals, arrays, pointers, sorting, and basic mathematics.

------------------------------------------------------------------------

## 📂 Files Included

### 1️⃣ bubble_sort.cpp

Demonstrates: - Sorting an array using Bubble Sort - Nested loops -
Swapping logic - Basic algorithm understanding

------------------------------------------------------------------------

### 2️⃣ control_statements.cpp

Demonstrates: - if statement - if-else statement - switch-case - Ternary
operator - goto statement - Basic control flow understanding

------------------------------------------------------------------------

### 3️⃣ factorial.cpp

Demonstrates: - for loop - Multiplicative accumulation - Mathematical
logic building

------------------------------------------------------------------------

### 4️⃣ for_while_dowhile.cpp

Demonstrates: - for loop - while loop - do-while loop - Differences
between loop structures

------------------------------------------------------------------------

### 5️⃣ gcd_lcm.cpp

Demonstrates: - Euclidean Algorithm - Functions - Mathematical logic -
Relationship between GCD and LCM

------------------------------------------------------------------------

### 6️⃣ io_streams.cpp

Demonstrates: - cin and cout - getline() - Basic input/output formatting

------------------------------------------------------------------------

### 7️⃣ loop_sum_input.cpp

Demonstrates: - Loop-based input - Accumulation pattern - Basic
arithmetic operations

------------------------------------------------------------------------

### 8️⃣ pointer_basics.cpp

Demonstrates: - Pointer declaration - Address operator (&) -
Dereferencing (\*) - Memory basics

------------------------------------------------------------------------

## 🧠 Concepts Covered

-   Variables and Data Types\
-   Control Statements\
-   Loops\
-   Functions\
-   Arrays\
-   Sorting Algorithms\
-   Mathematical Programs\
-   Pointers\
-   Input/Output Handling

------------------------------------------------------------------------

## 🛠 Compile & Run

Use g++ to compile:

g++ filename.cpp -o output ./output

Example:

g++ bubble_sort.cpp -o bubble ./bubble

------------------------------------------------------------------------

## 🎯 Learning Outcome

After completing these programs, you will:

✔ Understand basic control flow in C++\
✔ Be comfortable with loops and conditionals\
✔ Understand array manipulation\
✔ Learn basic algorithm implementation\
✔ Understand pointer fundamentals\
✔ Build strong problem-solving foundation

------------------------------------------------------------------------

These programs form the foundation for Data Structures and
Object-Oriented Programming 🚀
